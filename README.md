# A basic microsite for Greenpeace Planet 4 project.

![Planet4](./planet4.png)

## Run

Some basic npm scripts are included.

Start by switching to Node 16

```bash
nvm use 16
```

### Lint

```bash
npm run lint-js
npm run lint-css
```

### Build

```bash
npm run build
npm run static-dev
```

### Run

```bash
npm run server
```
